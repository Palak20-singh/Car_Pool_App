## Overview

The **Car Pool App** is a Java application that simulates a ride-booking and carpooling system. It allows users to act as drivers (who offer rides) or passengers (who book available rides), helping reduce travel costs and promote sustainable commuting.

- üßë‚Äç‚úàÔ∏è **Driver Registration** ‚Äî Drivers can register and offer rides with details like source, destination, and available seats

- üí∫ **Seat Management** ‚Äî Tracks available seats and updates capacity upon booking
- üìã **Ride Listing** ‚Äî View all available rides in the system

## üöÄ Getting Started

### Clone the Repository

```bash
git clone https://github.com/Palak20-singh/Car_Pool_App.git
cd Car_Pool_App
```

### Compile and Run (Command Line)

```bash
# Compile the source file
javac Ridebookingsystem.java

# Run the application
java Ridebookingsystem
```

### Run in Eclipse

1. Open intellij IDEA
2. Go to **File ‚Üí Import ‚Üí Existing Projects into Workspace**
3. Select the cloned `Car_Pool_App` folder
4. Click **Finish**
5. Right-click `Ridebookingsystem.java` ‚Üí **Run As ‚Üí Java Application**

## üíª Usage

Once the application starts, follow the on-screen menu prompts to:

1. **Offer a Ride** ‚Äî Enter your name, source, destination, and number of available seats
2. **Book a Ride** ‚Äî Browse available rides and confirm your booking
3. **View All Rides** ‚Äî See a list of all currently offered rides and their seat availability
4. **Cancel a Ride/Booking** ‚Äî Remove your ride offer or cancel a confirmed booking
5. **Exit** ‚Äî Close the application
